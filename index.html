Variables

Definition: Anything that can vary. JavaScript uses dynamic typing for variable which means a variable could have an integer, string, boolean, or float value and could be change anytime.

There are 2 ways to declare a variable in JavaScript

  1. let: This variable will have limited scope and only visible to the code block that it is in. Developers tend to use this more. 

              
            for (let i = 0; i < 2; i++) { 
              console.log(i); // 0, 1
            }

            console.log(i); // Reference Error

  2. var: This variable can be called anywhere. Over used of this could lead to variable name conflict which could be hard to track. 

              
            for (var i = 0; i < 2; i++) { 
              console.log(i); // 0, 1
            }

            console.log(i); // 1

User Input

We can get user input either by HTML elements like text box or simply by using prompt(). This is useful as it allows the program to interact with the user. 

      
        let x = prompt("Type your name in"); // Input: John
        alert("Hello " + x + "!"); // Output: "Hello John!"

Basic Math in JavaScript

Javascript could be used to perform mathematical expression. There are 5 mathematical operators in JavaScript:

  1. "+" operator: This operator is used to get the sum of two numbers

              
            let x = 1;
            x = x + 1;
            x; // 2

    Here are the short hands for writing the above code block:

              
            let x = 1;
            x += 1;
            x; // 2
          
        
              
            let x = 1;
            x++;
            x; // 2

  2. "-" operator: This operator is used to get the difference of two numbers

              
            let x = 1;
            x = x - 1;
            x; // 0

    Here are the short hands for writing the above code block:

              
            let x = 1;
            x -= 1;
            x; // 0
          
        
              
            let x = 1;
            x--;
            x; // 0

  3. "*" operator: This operator is used to get the multiplication of two numbers

              
            let x = 1;
            x = x * 2;
            x; // 2

    Here is the short hand for writing the above code block:

              
            let x = 1;
            x *= 2;
            x; // 2

  4. "/" operator: This operator is used to get the result of one number divided by the other 

              
            let x = 4;
            x = x / 2;
            x; // 2

    Here is the short hand for writing the above code block:

              
            let x = 4;
            x /= 2;
            x; // 2

  5. "%" operator: This operator is used to get the remainder of the result of one number divided by the other. This is useful to use when you want to know if the number is even or odd. 

              
            let x = 1;
            x = x % 2;
            x; // 1

    Here is the short hand for writing the above code block:

              
            let x = 1;
            x %= 2;
            x; // 1
